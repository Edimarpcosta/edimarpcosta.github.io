Formatação de Texto Inteligente

Como deixar o texto em negrito, itálico ou colorido?

O campo "Informação/Descrição" possui um assistente rápido. Selecione o texto que deseja alterar e clique com o botão direito do mouse sobre ele.

Um menu irá aparecer oferecendo as opções: Negrito, Itálico, Sublinhado, cores ou Títulos. O sistema colocará automaticamente os códigos de HTML em volta do seu texto.

Posso usar tags HTML manualmente?

Sim! Você pode escrever códigos HTML como:

  • <b>texto</b> para negrito
  • <h1>Título Maior</h1> para Títulos de aviso
  • <span style="color:red">Atenção</span> para textos com cores
  • <br> para forçar uma quebra de linha

Prazos e Validade

Como funciona a Validade/Prazo dos avisos?

Você deve preencher a data usando o formato numérico padrão brasileiro: DD/MM/YYYY ou DD/MM/AA. Exemplo: 25/12/2026.

Sempre que usuário abrir o Mural, o sistema verificará se essa data já passou (vencido). Se a data já tiver passado de "Hoje", esse card sairá da sua aba original e irá para a pasta especial de histórico chamada "Ações Anteriores (Sem Validade)".

Aba de Produtos em Destaque

Como eu puxo os produtos da planilha mãe?

Quando você estiver logado no Manager e selecionar o menu lateral "Produtos em Destaque", o formulário de "Novo Registro" apresentará um botão extra chamado "🛒 Buscar Produto".

Ao clicar nele, você poderá buscar pelo NOME ou SKU. Clicando sobre o produto na lista, o título e a imagem serão preenchidos automaticamente no formulário! Tudo o que você precisa fazer é escrever as regras do produto e salvar.

Links e Imagens

Como adicionar a imagem em um card?

Basta colar um link válido de uma imagem (jpg, png, gif) no campo "Link da Imagem". A imagem pode vir do Google Drive, AWS, WhatsApp (copiando url da imagem web), etc. Essa imagem será renderizada no topo do card com a opção de ampliar na lupa.

Como colocar um link dentro do texto?

Se você colar uma URL completa (exemplo: https://google.com) solta dentro do texto, ela já vai virar um link clicável automaticamente no mural. Se for um link de anexo, o ideal é colocá-lo no campo "Link de Acesso" do formulário para virar o botão azul do card.
