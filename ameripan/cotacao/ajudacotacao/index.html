Dados da Cotação

Preencha os dados do vendedor e do cliente. Esses dados aparecem no PDF e na mensagem de WhatsApp.

1
Nº Cotação e Data — editáveis no cabeçalho (canto superior direito). São incrementados automaticamente ao criar nova cotação.
2
Vendedor — seu nome. Aparece no cabeçalho do PDF e na mensagem WhatsApp.
3
WhatsApp Vendedor — número formatado automaticamente. Usado no botão WhatsApp e como link clicável no PDF.
4
Cliente / Cód. Cliente — razão social e código interno. Exibidos no bloco "CLIENTE / ENTREGA" do PDF.
5
Endereço / CNPJ e Condição de Pagamento — informações adicionais do bloco do cliente no PDF.
6
Exibir coluna Código de Barras — toggle para mostrar/ocultar a coluna EAN/GTIN na tabela e no PDF.
7
Histórico de Cotações — expanda para ver e restaurar as últimas 20 cotações salvas.
8
Nova Cotação — salva a cotação atual no histórico e inicia uma nova com ID incrementado.
Todos os campos são salvos automaticamente a cada alteração (localStorage). Ao fechar e reabrir o navegador a cotação é restaurada.

Adicionar / Editar Item

O formulário laranja permite adicionar produtos à cotação de duas formas:

Forma 1: Por Código

1
Digite o código do produto no campo "Código". O nome é preenchido automaticamente via catálogo.
2
Informe a quantidade e o preço unitário.
3
Pressione Enter no campo Preço ou clique no botão para adicionar.

Forma 2: Por Busca de Nome

1
Digite parte do nome do produto no campo "Descrição (busca)". Sugestões aparecem conforme você digita.
2
Clique na sugestão desejada — código e descrição são preenchidos automaticamente.
3
Opcionalmente, use "Adicionar todos" no topo da lista para inserir todos os resultados do filtro de uma só vez.

Cadastrar Novo Produto

Clique no botão Novo produto para abrir o modal de cadastro. Preencha o código, descrição, unidade e URL de imagem (opcional). Requer Apps Script configurado.

Ao clicar no de um item na tabela (em telas pequenas ou via botão Editar), o formulário muda para modo edição — borda azul tracejada e badge "EDITANDO ITEM". Após salvar, volta ao modo de adição.

Ações disponíveis na tabela

  • Editar — abre edição inline na própria linha (desktop) ou no formulário superior (mobile)
  • Remover — remove o item da cotação
  • Arrastar (Drag & Drop) — arraste as linhas para reordenar a cotação
  • Observações — campo de texto livre abaixo da tabela para anotações (aparecem no PDF)

Edição Inline v26

No desktop, há três formas de editar itens diretamente na tabela, sem abrir formulários:

Duplo Clique em Qtde ou Preço

Dê um duplo clique na célula de Qtde ou Preço Un para abrir uma entrada inline diretamente na célula. Pressione Enter para salvar ou Esc para cancelar. O total é atualizado instantaneamente.

As colunas Qtde e Preço possuem um ícone discreto no cabeçalho, indicando que são editáveis por duplo clique.

Edição Completa da Linha

Clique no botão ou no ícone que aparece ao passar o mouse sobre qualquer campo com lápis. A linha inteira vira um formulário de edição com todos os campos, incluindo código, descrição, unidade, barras, qtd e preço. Pressione Enter para salvar ou Esc para cancelar.

Drag & Drop para Reordenar

Clique e arraste qualquer linha da tabela para uma nova posição. A linha fica semitransparente durante o arraste e a posição de soltura é indicada por uma borda laranja. A nova ordem é salva automaticamente.

Em telas menores (mobile), a edição inline não está disponível. Use o botão Editar nos cards para abrir o formulário superior.

Gerenciamento de Imagens

As imagens dos produtos são carregadas automaticamente conforme a fonte configurada. Você também pode editar e definir imagens manualmente.

Menu Contextual de Imagem

Clique com o botão direito (desktop) ou faça um toque longo (mobile) na miniatura de um produto para abrir o menu contextual:

Definir URL da imagem
Abrir imagem
Cadastrar novo item
  • Definir URL da imagem — abre modal para inserir ou substituir a URL. O sistema avisa se o produto já possui imagem. Após confirmar, a URL é gravada na Col. C da planilha via Apps Script e o cache é atualizado.
  • Abrir imagem — abre a URL da imagem atual em nova aba (disponível apenas quando há imagem).
  • Cadastrar novo item — atalho para o modal de cadastro de produto novo.

Placeholder sem imagem

Quando um produto não tem imagem, exibe um ícone cinza. Clique com o botão direito para definir a URL manualmente.

Imagens são cacheadas em memória durante a sessão. Para recarregar uma imagem atualizada, faça F5 na página.

Exportação (PDF / Excel / WhatsApp)

Gerar PDF

Gera um PDF profissional em formato paisagem (A4) com:

  • Cabeçalho com logo Ameripan, título e número/data da cotação em destaque
  • Bloco CLIENTE / ENTREGA e bloco DADOS COMERCIAIS lado a lado
  • Tabela de itens com imagens (quando disponíveis), código de barras opcional
  • Total geral, campo de observações e linha de assinatura
  • Link do WhatsApp clicável no PDF
  • Paginação automática com mini-header nas páginas seguintes
  • Rodapé com nome empresa, número da cotação e data em todas as páginas
Dica: Ajuste a Altura das linhas no PDF nas Configurações (10 a 40 mm) para controlar o tamanho das imagens na tabela.

Exportar Excel

Gera um arquivo .xlsx com cabeçalho de dados da cotação e tabela completa de itens (código, descrição, unidade, qtde, preço, total e total geral).

WhatsApp

Abre o WhatsApp Web com mensagem formatada contendo todos os itens, preços e total. Usa o número do vendedor preenchido nos dados da cotação.

Copiar

Copia o texto formatado da cotação para a área de transferência para colar em qualquer aplicativo.

No mobile, os botões de ação ficam na barra inferior fixa (FAB): XLS, Copiar, WhatsApp e PDF.

Fonte de Imagens

  • GitHub — busca imagens do repositório (mais rápido/estável). Formato: {base}/{código}.jpg ou .png
  • Site Ameripan — scraper que busca a imagem diretamente no site ameripan.com.br
  • Planilha (Col. C) — usa a URL de imagem salva na coluna C da planilha (padrão)

Apps Script URL

URL do Web App do Google Apps Script para gravar dados na planilha (imagens, novos produtos, CSV). Sem ela, as funções de gravar imagem, cadastrar produto e upload CSV ficam desabilitadas.

Altura das linhas no PDF

Controla a altura de cada linha de produto no PDF (10–40mm). Linhas maiores = imagens maiores. Linhas menores = mais itens por página.

Importante: Ao alterar a "Fonte de Imagens" e salvar, o cache de catálogo e imagens é limpo e o catálogo é recarregado automaticamente.

Auto-Enriquecimento de Imagens

Varre todos os produtos do catálogo sem imagem na planilha, busca nas fontes disponíveis e grava os links encontrados automaticamente na coluna C.

Como Usar

1
Abra as Configurações () e confirme que Apps Script Web App URL está preenchido.
2
Dentro das Configurações, expanda "Auto-Enriquecimento de Imagens" e escolha o modo:
GitHub + Site → Busca primeiro no GitHub, depois no site Ameripan como fallback
Scanear Site Ameripan → Busca exclusivamente no site ameripan.com.br
3
Acompanhe o progresso pela barra de progresso, pelos contadores (Varridos / Encontrados / Salvos / Erros) e pelo log em tempo real.

Scanear Site Ameripan — Como funciona

1
Para cada produto sem imagem, o sistema busca em ameripan.com.br/buscar?q={código}
2
O HTML da página é parseado para localizar o produto com o SKU exato.
3
Se encontrado, extrai a URL da imagem e grava automaticamente na Col. C da planilha.

O scraper tenta múltiplas vias CORS: fetch direto → allorigins.win → corsproxy.io.

GitHub: processado em lotes de 5 itens. Site Ameripan: lotes de 3. Para catálogos grandes pode levar vários minutos.

Upload CSV em Massa v26

Envie um arquivo .csv para atualizar imagens e/ou cadastrar produtos em massa na planilha via Apps Script.

Formatos aceitos

  • 2 colunas (cod, linkimg) — atualiza apenas a URL de imagem do produto existente.
  • 3 colunas (cod, produto, linkimg) — atualiza nome + imagem se o produto já existir; insere como novo se não existir.

Como usar

1
Abra as Configurações e expanda "Upload CSV em Massa".
2
Clique em Escolher arquivo e selecione seu .csv ou .txt.
3
Clique em Processar CSV. Acompanhe o status e o relatório de resultados.
O CSV pode ter cabeçalho (primeira linha ignorada se não for numérica). Separador: vírgula ou ponto-e-vírgula.
Requer Apps Script URL configurado nas Configurações. Sem ele, o botão Processar CSV ficará desabilitado.

Dicas e Boas Práticas

  • 🔄 Cache de catálogo: válido por 24h. Para forçar recarga, salve as Configurações.
  • 📱 Responsivo: no mobile, use os cards de itens e a barra FAB de ação fixa na parte inferior.
  • 🔒 Dados locais: tudo salvo no localStorage do navegador. Não enviado para servidores externos.
  • 📋 Histórico: as últimas 20 cotações ficam salvas. Expanda "Histórico de Cotações" na seção Dados para recuperar.
  • 🖼️ Imagens no PDF: para melhores resultados use a fonte "GitHub" que tem CORS habilitado.
  • ➕ Novo produto: clique em "Novo produto" no formulário para cadastrar um item inexistente na planilha (requer Apps Script).
  • 🖱️ Duplo clique: o modo mais rápido para corrigir quantidade ou preço de um item já adicionado.
  • 📥 CSV em massa: ideal para importar um lote de imagens de uma vez antes de gerar PDFs.
  • ❓ Ajuda: acesse este guia a qualquer momento pelo ícone no cabeçalho.
