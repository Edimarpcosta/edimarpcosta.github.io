Visão Geral do Sistema

O Ameripan Cotação é um sistema web completo para criar, editar e exportar cotações de produtos. Funciona 100% no navegador — sem necessidade de instalação.

Principais Funcionalidades

  • Catálogo automático — carrega produtos diretamente da planilha Google Sheets
  • Busca inteligente — busca por nome com sugestões em tempo real
  • Imagens automáticas — busca imagens no GitHub, Site Ameripan ou planilha
  • PDF profissional — gera PDF com layout corporativo e imagens
  • WhatsApp — envia cotação formatada direto pelo WhatsApp
  • Excel — exporta para planilha .xlsx
  • Histórico — salva e recupera cotações anteriores
  • Drag & Drop — reorganize itens arrastando as linhas
  • Enriquecimento — varre o catálogo e salva imagens automaticamente NOVO v25

Dados da Cotação

Preencha os dados do vendedor e do cliente. Esses dados aparecem no PDF e na mensagem de WhatsApp.

1
Vendedor — seu nome. Aparece no cabeçalho do PDF.
2
WhatsApp Vendedor — número formatado automaticamente. Usado no botão "WhatsApp" e no link do PDF.
3
Cliente / Cód. Cliente — razão social e código interno. Exibidos no bloco "Cliente" do PDF.
4
Endereço / CNPJ e Condição de Pagamento — informações adicionais para o bloco do cliente.
5
Exibir coluna Barras — toggle para mostrar/ocultar a coluna de Código de Barras (EAN/GTIN) na tabela e no PDF.
6
Nova Cotação — salva a cotação atual no histórico e inicia uma nova com ID incrementado.
Todos os campos são salvos automaticamente a cada alteração (localStorage). Ao fechar e reabrir o navegador, a cotação é restaurada.

Adicionar / Editar Item

O formulário permite adicionar produtos à cotação de duas formas:

Forma 1: Por Código

1
Digite o código do produto no campo "Código". O sistema busca automaticamente o nome na planilha.
2
Informe a quantidade e o preço unitário.
3
Pressione Enter ou clique no botão para adicionar.

Forma 2: Por Busca de Nome

1
Digite parte do nome do produto no campo "Descrição (busca)". Sugestões aparecem conforme você digita.
2
Clique na sugestão desejada. Código e descrição são preenchidos automaticamente.
3
Opcionalmente, use o botão "Adicionar todos" no topo da lista de sugestões para inserir todos os resultados do filtro de uma vez.

Exportação (PDF / Excel / WhatsApp)

Gerar PDF

Gera um PDF profissional em formato paisagem (A4) com:

  • Cabeçalho com logo Ameripan e número da cotação
  • Blocos de dados do cliente e dados comerciais
  • Tabela de itens com imagens (quando disponíveis)
  • Total geral, linha de assinatura e botão WhatsApp clicável
  • Paginação automática com mini-header nas páginas seguintes
Dica: Na janela de Configurações, ajuste a altura das linhas no PDF (10mm a 40mm) para controlar o tamanho das imagens.

Exportar Excel

Gera um arquivo .xlsx com dados do cabeçalho e tabela completa de itens.

WhatsApp

Abre o WhatsApp Web com uma mensagem formatada contendo todos os itens, preços e total. Usa o número do vendedor preenchido nos dados.

Copiar

Copia o texto formatado da cotação para a área de transferência, permitindo colar em qualquer aplicativo.

Fonte de Imagens

  • GitHub — busca imagens do repositório (mais rápido e estável). Formato: {base}/{código}.jpg ou .png
  • Site Ameripan — scraper que busca a imagem diretamente do site ameripan.com.br (pode ser mais lento)
  • Planilha (Col. C) — usa a URL de imagem salva na coluna C da planilha de produtos

Apps Script URL

URL do Web App do Google Apps Script que permite gravar dados na planilha (imagens, novos produtos). Sem ela, as funções de gravar imagem e cadastrar produto ficam desabilitadas.

Importante: Ao alterar a "Fonte de Imagens" e salvar, o cache de catálogo e imagens é limpo e o catálogo é recarregado.

Auto-Enriquecimento de Imagens

O enriquecimento automático varre todos os produtos do catálogo que não têm imagem na planilha, busca as imagens nas fontes disponíveis, e grava os links encontrados automaticamente na coluna C da planilha.

Como Usar

1
Abra as Configurações (ícone ).
2
Verifique que o campo Apps Script Web App URL está preenchido.
3
Na seção "Auto-Enriquecimento de Imagens", escolha um dos botões:
GitHub + Site → Busca primeiro no GitHub, depois no site Ameripan
Scanear Site Ameripan → Busca exclusivamente no site ameripan.com.br
4
Acompanhe o progresso na barra de progresso, nos contadores de estatísticas e no log de atividade.

Scanear Site Ameripan NOVO v25

A funcionalidade "Scanear Site Ameripan" é uma novidade da versão 25. Ela varre exclusivamente o site ameripan.com.br buscando imagens de produtos que ainda não possuem URL na planilha.

Como Funciona?

1
Para cada código de produto sem imagem, o sistema faz uma busca no site: ameripan.com.br/buscar?q={código}
2
O HTML da página é parseado para encontrar o produto com o SKU exato correspondente.
3
Se encontrado, extrai a URL da imagem e o nome do produto.
4
As imagens encontradas são gravadas automaticamente na coluna C da planilha via Apps Script.

Proxy CORS (v25)

O scraper tenta múltiplas vias para contornar bloqueios CORS:

  • Direto — fetch direto ao site (funciona em certas configurações)
  • allorigins.win — proxy CORS público como fallback
  • corsproxy.io — segundo fallback CORS
Atenção: O scan pelo site é mais lento que o GitHub (processa em lotes de 3). Para catálogos grandes, pode levar alguns minutos. Acompanhe pelo log em tempo real.

Dicas e Boas Práticas

  • 🔄 Cache inteligente: O catálogo é cacheado por 24h. Para forçar recarga, salve as configurações.
  • 📱 Responsivo: O sistema funciona bem em celulares com cards de itens e barra de ações fixa.
  • 🔒 Tudo local: Seus dados são salvos no navegador (localStorage). Não são enviados para servidores externos.
  • 📋 Histórico: As últimas 20 cotações ficam salvas. Use a seção "Histórico de Cotações" para recuperar.
  • 🖼️ Imagens no PDF: Para melhores resultados, use a fonte "GitHub" que tem CORS habilitado.
  • ➕ Novo produto: Clique em "Novo produto" no formulário para cadastrar um item que não existe na planilha.
