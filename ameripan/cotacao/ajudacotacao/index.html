Tutorial Cotação Ameripan v24

O Sistema de Cotação Ameripan é uma ferramenta open-source desenvolvida para otimizar o fluxo de vendas. Ele centraliza a busca de produtos, permite digitação super rápida via teclado e gera orçamentos profissionais em PDF, Excel ou mensagens diretas no WhatsApp.

Acessar o Sistema de Cotação
Chamada para Colaboração: As imagens deste tutorial são representações simuladas. Sinta-se à vontade para fazer um Pull Request no repositório adicionando prints reais das telas para ajudar a comunidade!

Passo 1: Configurações Iniciais

Antes de iniciar suas cotações, clique no ícone de engrenagem para ajustar as integrações do sistema.

  • Fonte de Imagens: Escolha entre GitHub (mais rápido), Scraper ou Planilha.
  • Apps Script URL: Insira a URL do seu script para gravar dados.
Tela de Configurações [X]
